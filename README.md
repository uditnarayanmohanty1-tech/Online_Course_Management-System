# 📚 Online Course Management System

## 📌 Project Description
The Online Course Management System is a Java-based console application developed using Core Java and Object-Oriented Programming (OOP) concepts.

This system allows students to enroll in courses, track their progress, and view course details. It simulates a basic e-learning platform and helps in understanding real-world application structure.

---

## 🎯 Objectives
- Manage course information  
- Manage student details  
- Enroll students into courses  
- Track student progress  
- Apply OOP concepts  

---

## 🛠️ Technologies Used
- Java (Core Java)  
- OOP Concepts  
- VS Code / Eclipse  

---

## 📂 Project Structure
OnlineCourseManagement/
│── Course.java
│── Student.java
│── CourseSystem.java

---

## ⚙️ How to Run the Project

1. Open terminal in project folder  
2. Compile the code:
   javac *.java
3. Run the program:
java CourseSystem
